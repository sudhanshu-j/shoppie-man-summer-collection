## 🌟 Overview

**Shoppie - Man Summer Collection** is a responsive e-commerce website template, designed with a focus on user experience and modern UI. It consists of three core components:

- **HTML (index.html)**: Provides the structure for the webpage, including sections like the Hero, Features, Offers, and Footer.

- **CSS (style.css)**: Adds the style, implementing responsive design with media 
queries, custom variables, and reusable design patterns.

- **JavaScript (script.js)**: Enhances the interactivity and functionality of the website, including dynamic behaviors like navigation toggling and product interactions.

The goal is to provide a **visually appealing**, **responsive**, and **user-friendly** website that looks amazing on every device—from mobile to desktop.

---

## ✨ Features

- 📱 **Fully Responsive Layout**: The website automatically adjusts for different screen sizes using media queries.

- 🛒 **Product Showcase**: Beautifully displays products from the "Man Summer Collection" with an interactive and user-friendly layout.

- 🎨 **Modern and Clean Design**: Sleek, minimal design with generous spacing for offers, descriptions, and call-to-action buttons.

- 💡 **Interactive UI**: Hover effects, button animations, and smooth transitions.

- 🌐 **Cross-Browser Compatible**: Supports all modern browsers, ensuring 
accessibility and consistency across devices.

- 📑 **Semantic HTML Structure**: Utilizes clear HTML tags like section, header, footer, and article for better SEO and organization.

---

## ⚙️ Technologies Used

- **HTML5**: For structuring the content.

- **CSS3**: For styling and implementing responsive design.

- **JavaScript**: For interactivity (navigation toggle, product interactions, etc.).

- **CSS Variables**: To facilitate easy theme customization.

- **IonIcon**: For clean and scalable icons.

- **Google Fonts**: The `Space Grotesk` font is used for modern typography.

## 💡 Usage

- 📱 **Responsive Design**: The website adjusts for various screen sizes using **media queries**:
  
  - **Mobile**: `min-width: 575px`
  
  - **Tablets**: `min-width: 768px`
  
  - **Desktops**: `min-width: 992px+`

- 🎨 **CSS Custom Properties (Variables)**:  
  The project utilizes **CSS variables** to manage common design elements such as colors, fonts, and spacing.  
  **Example variables**:
  
  - `--bg-summer-sky`: The background color for the hero section.
  
  - `--fontSize-4`: Font size for subheadings or product descriptions.

- 🧑‍💻 **Media Queries**:  
  The CSS file ensures the layout is adaptable for mobile, tablet, and desktop views.

- 🛍️ **Product Interactivity**:  
  The **`script.js`** file controls smooth transitions for adding items to the cart, as well as hover and button animations.

## 📄 Stylesheet Analysis

### General Design & Layout:
- **Hero Section**:  
  The `.hero` class defines a captivating hero section with a vibrant background color (`--bg-summer-sky`) and centered text for maximum impact. 🌅

- **Feature Section**:  
  Key product features are displayed in a grid layout, with ample spacing for easy readability. Action buttons encourage user interaction. 📦

- **Offer Cards**:  
  Each offer card in the `.offer` section includes a subtle shadow effect to create depth and increase focus. The layout adapts to different screen sizes for better presentation. 🏷️

- **Footer**:  
  The footer includes social media links, quick links, and hover effects on icons for smooth interactions. 📱

- **Responsive Design**:  
  The layout adapts from a single-column design on mobile to multi-column designs on larger screens, ensuring content is neatly organized. 📐
